• The CERES Emulator is a Python3+PyQt5 "Device" that helps development and testing of the CERES Platform.
  • The emulator is fully open sourced and there is a reference implementation of the CERES server coming soon.
  • This is a particularly useful tool when testing new spectrums or mesh control strategies, as the emulator can stand in lieu of full NX3 fixtures in production as a testing and development tool.
